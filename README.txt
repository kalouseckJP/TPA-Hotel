El programa solo funcionara si se tiene java 8 como compilador, en consola revisar version con "javac -version".

De no tener java 8 como compilador pero si instalado, ir a "variables de entorno" y tanto variables de usuario como de sistema eliminar las carpetas de Java o Oracle que no especifiquen java 8.

Teniendo java 8 como compilador, en consola desde la carpeta bas "GUIS" ingresar "javac ./hotel/Main.java"

Para ejecutar "java hotel.Main"
